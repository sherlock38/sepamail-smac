========================
IMPORTANT
========================

Modify the application configuration file to suit your directory structure
before starting the SMAC service.


========================
STARTING THE SMAC DAEMON
========================

To start the Smac daemon from the command line, go to the root directory of
the Smac daemon and type the following:

./smac.sh start


========================
STOPPING THE SMAC DAEMON
========================
To stop the Smac daemon from the command line, go to the root directory of 
the Smac daemon and type the following:

./smac.sh stop
